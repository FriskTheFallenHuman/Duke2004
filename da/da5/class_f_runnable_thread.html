Detailed Description

This is the base interface for all runnable thread classes. It specifies the methods used in managing its life cycle.

◆ Kill()

virtual UBOOL FRunnableThread::Kill ( UBOOL  bShouldWait = FALSE)
pure virtual

Tells the thread to exit. If the caller needs to know when the thread has exited, it should use the bShouldWait value and tell it how long to wait before deciding that it is deadlocked and needs to be destroyed. The implementation is responsible for calling Stop() on the runnable. NOTE: having a thread forcibly destroyed can cause leaks in TLS, etc.

Parameters
bShouldWaitIf true, the call will wait for the thread to exit
Returns
True if the thread exited graceful, false otherwise
