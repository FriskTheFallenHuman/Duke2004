void FViewportConfig_Data::ToggleMaximize ( FViewport *  Viewport)

Either resizes all viewports so that the specified Viewport is fills the entire editor window, or restores all viewports to their previous sizes. WxEditorFrame will lock all splitter drag bars when a viewport is maximized. This function is called by clicking a button on the viewport toolbar.
