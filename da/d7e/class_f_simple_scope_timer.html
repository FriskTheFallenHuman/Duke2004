Detailed Description

The simple scope timer is an easy way to look for slow parts in a large function. Place a stack-local instance in each scope, and any that exceed the constructed threshold will warn using the provided filename / line number.

The timer isn't super-lightweight, it copies the filename string when constructed and immediately warns on ones that exceed the threshold.

It's intended to be used for one-off optimizations, not left in and enabled always.
