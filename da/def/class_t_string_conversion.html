Detailed Description

template<typename CONVERT_TO, typename CONVERT_FROM, typename BASE_CONVERTER, DWORD DefaultConversionSize = 128>
class TStringConversion< CONVERT_TO, CONVERT_FROM, BASE_CONVERTER, DefaultConversionSize >

Class takes one type of string and converts it to another. The class includes a chunk of presized memory of the destination type. If the presized array is too small, it mallocs the memory needed and frees on the class going out of scope. Uses the overloaded cast operator to return the converted data.
