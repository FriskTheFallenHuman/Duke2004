◆ CleanupDataForDeferredDeletion()

void FParticleDynamicData::CleanupDataForDeferredDeletion ( )
inline

Called when a dynamic data is being deleted when double-buffering. This function is responsible for releasing all non-render resource data. The complete deletion must be deferred to ensure the render thread is not utilizing any of the resources.
