UINT FSkeletalMeshFragmentCookOutputInfo::NumPartitions

On the PS3, libedgegeom partitions each mesh fragment into a number of SPU-sized partitions, each of which will be processed by a single SPURS job when it is rendered. The per-partition data is stored here.
