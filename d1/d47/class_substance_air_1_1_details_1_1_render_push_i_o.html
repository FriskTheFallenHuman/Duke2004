Constructor from push I/O to duplicate.

Parameters
renderJobParent render job that owns this instance
srcThe canceled push I/O to copy
dupDuplicate job context Build a push I/O from a canceled one and optionnaly filter outputs. Push again SRC render tokens (of not filtered outputs).
Warning
Resulting push I/O can be empty (hasOutputs()==false) if all outputs are filtered OR already computed. In this case this push IO is no more usefull and can be removed.
