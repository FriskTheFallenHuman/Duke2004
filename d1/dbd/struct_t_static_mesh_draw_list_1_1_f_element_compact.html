Detailed Description

template<typename DrawingPolicyType>
struct TStaticMeshDrawList< DrawingPolicyType >::FElementCompact

This structure stores the info needed for visibility culling a static mesh element. Stored separately to avoid bringing the other info about non-visible meshes into the cache.
