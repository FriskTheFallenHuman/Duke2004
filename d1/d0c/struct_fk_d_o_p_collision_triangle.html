Detailed Description

template<typename BOUND_TYPE>
struct FkDOPCollisionTriangle< BOUND_TYPE >

Used to pad and align bounding boxes on the stack so that SIMD can use them Represents a single triangle. A kDOP may have 0 or more triangles contained within the node. If it has any triangles, it will be in list (allocated memory) form.
