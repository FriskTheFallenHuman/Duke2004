The meat of the filter. Filters are provided with pointers to the beginning and end of a character buffer. They return a result buffer built by "processing" the contents of the input buffer. Processing has different meaning for different filters.

Implemented in FSequencedTextFilter, FCommentStrippingFilter, and FMacroProcessingFilter.
