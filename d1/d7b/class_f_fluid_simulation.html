◆ SimulationIndex

INT FFluidSimulation::SimulationIndex
protected

Used to double buffer data for thread-safe interactions between the rendering thread and the simulation thread. The simulation thread always writes to [SimulationIndex] and the rendering thread reads from [1 - SimulationIndex]. SimulationIndex is flipped in RenderThreadTick(), after the rendering thread has blocked on the simulation thread.
