Detailed Description

This struct holds the information used to do a box and/or point check against the terrain BV tree. It is used purely to gather multiple function parameters into a single structure for smaller stack overhead.

Sets up the FTerrainBVTreeBoxCollisionCheck structure for performing swept box checks against a kDOPTree. Initializes all of the variables that are used throughout the check.

InStart– The starting point of the trace
