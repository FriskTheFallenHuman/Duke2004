Detailed Description

Copyright 1998-2013 Epic Games, Inc. All Rights Reserved.

FCallbackDeviceEditor.h: Callback device specifically for UnrealEd This device forwards events to objects that have registered with it. It maintains a list of Observers for each type of ECallbackEventType that is defined.
