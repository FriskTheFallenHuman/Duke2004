const TMap< AActor *, FIntPoint > *  ActorToUserDrawPosMap = NULL

See if any actors in ChildActors are a direct child of the Actor of this DrawInfo. If so, create a ChildInfo entry For each newly created ChildIndo entry, call this function on it in turn, to build out the whole tree

Parameters
ChildActorsThe actors to check for direct children
ActorToUserDrawPosMapOptional mapping of actors to their position in the graph.
This is used to preserve moved draw infos in the editor when they are destroyed during a refresh
