Detailed Description

Include the current implementation of a FBoneAtom, depending on the vector processing mode useful for get a stable readable ms (indirectly an fps counter) printout e.g. FStableSample Stable(0.2f); gives a stable sample for 0.2secs, assuming the values are seconds
