Detailed Description

template<typename COLL_DATA_PROVIDER, typename KDOP_IDX_TYPE, typename TREE_TYPE>
struct TkDOPBoxCollisionCheck< COLL_DATA_PROVIDER, KDOP_IDX_TYPE, TREE_TYPE >

This struct holds the information used to do a box and/or point check against the kDOP tree. It is used purely to gather multiple function parameters into a single structure for smaller stack overhead.

Sets up the TkDOPBoxCollisionCheck structure for performing swept box checks against a kDOPTree. Initializes all of the variables that are used throughout the check.

Parameters
InStart– The starting point of the trace
InEnd– The ending point of the trace
InExtent– The extent to check
InTraceFlags– The trace flags that might modify searches
InCollDataProvider– The struct that provides access to mesh/primitive specific data, such as L2W, W2L, Vertices, and so on
InResult– The out param for hit result information
