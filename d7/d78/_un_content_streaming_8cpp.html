◆ GMinimumStreamingCameraToMeshDistance

FLOAT GMinimumStreamingCameraToMeshDistance = 0.0f

Minimum allowed distance squared from the camera to the target mesh for the purposes of streaming. If this is less than one, the normal rules apply and when the camera is less than 1 unit from the the mesh, ALL mip levels are request. For tweaking via GOverallTextureStreamingBias, it is helpful to have this number larger than 1 so that control of all textures is possible.

◆ GNeverStreamOutTextures

UBOOL GNeverStreamOutTextures = FALSE

For testing, finding useless textures or special demo purposes. If TRUE, textures will never be streamed out (but they can be GC'd). Caution: this only applies to unlimited texture pools (i.e. not consoles)

◆ GOverallTextureStreamingBias

FLOAT GOverallTextureStreamingBias = 1.0f

Bias applied for ALL mip calculations. 1.0 = standard logic. < 1.0 = less texture resultion, > 1.0 more texture resolution. Caution: this is probably not useful with limited texture pools, which try to balance this automatically, and dynamically.

◆ GShadowmapStreamingFactor
