Detailed Description

Helper struct to ensure that secondary views (e.g. split-screen) renders the final image in the same SceneColorLDR buffer as the main view (view0). This is done by making sure secondary views always do an even number of ping-pongs, by counting them and let the first effect that renders to SceneColorLDR skip the bufferswap if it's an odd number.
