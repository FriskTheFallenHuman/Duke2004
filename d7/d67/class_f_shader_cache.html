◆ CompressedCache

TRefCountPtr<FCompressedShaderCodeCache> FShaderCache::CompressedCache
protected

Reference to the compressed shader cache corresponding to this shader cache. With seekfree loading on consoles, shader caches cooked into seekfree packages will be GC'ed shortly after the package is loaded, But the FCompressedShaderCodeCache will not be deleted as it will still be referenced by whichever FMaterialShaderMaps came from that shader cache. If this is the global shader cache, this is the only reference to the global FCompressedShaderCodeCache.
