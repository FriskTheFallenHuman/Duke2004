◆ UpdateMinDesiredLODLevel()

void FSkeletalMeshObject::UpdateMinDesiredLODLevel ( const FSceneView *  View,
const FBoxSphereBounds &  Bounds,
INT  FrameNumber 
)

Given a set of views, update the MinDesiredLODLevel member to indicate the minimum (ie best) LOD we would like to use to render this mesh. This is called from the rendering thread (PreRender) so be very careful what you read/write to.

Given a set of views, update the MinDesiredLODLevel member to indicate the minimum (ie best) LOD we would like to use to render this mesh. This is called from the rendering thread (PreRender) so be very careful what you read/write to. If this is the first render for the frame, will just set MinDesiredLODLevel - otherwise will set it to min of current MinDesiredLODLevel and calculated value.
