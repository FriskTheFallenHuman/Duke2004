FVector2D &  OutPreNormalizeUVExtents 
)

Compare function used to sort building scopes by their size. Larger scopes will come first. Given the scopes for a building, generate an atlas of texture subareas to fit all the scopes into a single texture

Parameters
ScopesList of scopes (faces) to make a single mesh out of
ScopeInfosList of associated infos for each face
bAllowCroppingTRUE if we should crop UVs to rectangular textures if needed
OutUVsOutput array (matching Scopes array) that will contain the offset/size of the faces in UV space (only written to if successful)
OutTextureSizeInWorldUnitsHow large the texture is in world units (largest dimension)
OutPreNormalizeUVExtentsMaximum UV extents, BEFORE the UVs were renormalized
Returns
TRUE if a valid layout was found
