Detailed Description

A set of the skeletal mesh vertices which use the same set of <MAX_GPUSKIN_BONES bones. In practice, there is a 1:1 mapping between chunks and sections, but for meshes which were imported before chunking was implemented, there will be a single chunk for all sections.
