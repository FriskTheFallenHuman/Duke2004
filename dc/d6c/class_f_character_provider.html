Detailed Description

This class represents objects that contain text which will be emitted into the output buffer (post-processed uc file). Can be a .uc file, include file, macro or macro parameter.

◆ ParseMacroParameterValues()

void FCharacterProvider::ParseMacroParameterValues ( const TArray< FString > *  ParamNames,
TArray< FString > *  ParamValues 
)
inline

Replaces macro parameter names in the macro literal with the actual values specified when the macro was invoked. Macros should always have at least one name/value pair; in cases where the macro has no parameters (such as macro parameters themselves), it should have a key/value pair that gives the name of the macro and the definition of that macro, or in the case of a macro parameter, the value assigned to that macro parameter.

Parameters
ParamNamespointer to an array of parameter names for the macro which corresponds to this CharacterProvider
ParamValuespointer to an array of parameter values for the macro which corresponds to this CharacterProvider

FString FCharacterProvider::PriorityStack
private

a stack of characters which is always processed before the main input buffer. Typically contains characters that have been returned via UngetChar(). Is also used to manually inject text into the output stream. (all insertions are reversed so insertion/deletion takes place at the END of the PriorityStack).
