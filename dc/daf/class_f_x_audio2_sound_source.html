◆ GetDestinationVoiceIndexForEffect()

INT FXAudio2SoundSource::GetDestinationVoiceIndexForEffect ( SourceDestinations  Effect)
protected

Utility function for determining the proper index of an effect. Certain effects (such as: reverb and radio distortion) are optional. Thus, they may be NULL, yet XAudio2 cannot have a NULL output voice in the send list for this source voice.

Returns
The index of the destination XAudio2 submix voice for the given effect; -1 if effect not in destination array.
Parameters
EffectThe effect type's (Reverb, Radio Distoriton, etc) index to find.
