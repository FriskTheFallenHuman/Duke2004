Detailed Description

This class is used for encapsulating tasks performed when the value of a property is changed outside of an editor property window. It creates and cleans up an archetype propagation archive to ensure that changes made to archetype objects are propagated to instances of that archetype, and sends Pre/PostEditChange notifications to the modified object.
