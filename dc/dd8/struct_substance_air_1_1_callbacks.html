Detailed Description

Abstract base structure of user callbacks Inherit from this base struct and overload the virtual methods to be notified by framework callbacks.

Use concrete callbacks instance in Renderer::setCallbacks().

Warning
Callback methods can be called from a different thread, not necessary the user thread (that call Renderer::run()).
