void FStructPair::PerformComparison ( struct FComparisonResults &  Result)

Compares the fields and data values for the two structs.

Parameters
Resultreceives the results of the comparison.

Performs a comparison of two versions of the same struct or class, looking for structs which were added, removed, or have functions which are different. When a difference is detected, the relevant data about the struct is put into the appopriate list.
