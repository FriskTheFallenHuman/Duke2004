AllowMoveToNextEdge this function is called from GetNextMoveLocation and allows specialized edges to keep getnextmovelocation from moving to the next edge in line when the edge's special behavior has not been triggered yet (e.g. an edge which trips an anim or some such)

Parameters
PathParams- path param struct associated with the entity being processed
bInPoly0- are we in poly0 of ttis edge?
bInPoly1- are we in poly1 of this edge?
Returns
TRUE if it's OK to move to the next edge
