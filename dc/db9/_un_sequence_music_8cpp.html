Helper function to check if either the mobile build directory (ex: Game\Build\IPhone or Game\Build\Android) exists, or if specified, the MP3 file within the Resources\Music folder of the mobile build directory

Parameters
MobileBuildDir- The mobile build directory to look for/in
MP3Filename- The name of the mp3 file to check for. If NULL, only check the directory
Returns
- Whether or not the directory or file was found
Note
- The FString comparison is not case sensitive. If the MP3Filename is found, it will be renamed to match the capitalization of the actual file name

◆ VerifyMP3File()

void VerifyMP3File ( FString  SeqPathName,
FString *  MP3Filename,
UBOOL  bPopupWarning = FALSE 
)

Issues a warning if the mp3 file is not found in the Resources\Music folder of any mobile build directories that exist for the game. Also issues a warning if the mp3 file is found in multiple mobile build and the capitalization does not match between the file names in the different directories. If the MP3Filename is found, it is renamed to match the capitalization of the actual file name since the mobile file systems are case sensitive.

Parameters
MP3Filename- The name of the mp3 file to look for
bPopupWarning- Whether to show the warning using a pop-up message (TRUE) or through the logs (FALSE)
