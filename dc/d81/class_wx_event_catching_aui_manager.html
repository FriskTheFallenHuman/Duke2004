Detailed Description

WxEventCatchingAuiManager

A simple wrapper class around wxAuiManager that allows us to intercept events from AUI. Normally we wouldn't need this, but AuiManager doesn't respect custom event handlers applied to either the manager itself or the associated frame window. Boo!
