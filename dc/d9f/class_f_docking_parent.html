Detailed Description

Abstract class that encapsulates wxWidgets docking functionality. Should be inherited by frames that wish to have dockable children.

◆ LoadDockingLayout()

UBOOL FDockingParent::LoadDockingLayout ( )

Loads the docking window layout from an XML file. This should be called after all docking windows have been added. This function checks to see if the docking parent version number stored in the INI file is less than the current version number. If it is, it will not load the settings and let the new defaults be loaded instead.

Returns
Whether or not a layout was loaded, if a layout was NOT loaded, the calling function should set default docking window positions.
