Detailed Description

template<typename T>
struct TContainerTraitsBase< T >

Helper for array traits. Provides a common base to more easily refine a portion of the traits when specializing. NeedsConstructor/NeedsDestructor is mainly used by the contiguous storage containers like TArray.
