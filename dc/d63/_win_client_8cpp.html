◆ GUseNewMouseInput

UBOOL GUseNewMouseInput = TRUE

Enable the new DirectInput mouse method by default. Toggle by exec "UseNewMouseInput" at run-time. It merges all mouse events for a whole frame into one single event and uses the frame delta. The old method was using the delta time since the mouse was last moved, which could be many many seconds.
