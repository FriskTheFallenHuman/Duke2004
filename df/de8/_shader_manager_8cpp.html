◆ ShouldInitShadersOnDemand()

static UBOOL ShouldInitShadersOnDemand ( )
static

When enabled, shaders will be initialized individually when needed for rendering, instead of all the shaders in a material shader map getting initialized when the material is loaded. This is useful for working around Nvidia drivers running out of paged pool memory, but as a result there will be a hitch the first time a new shader is needed for rendering.
