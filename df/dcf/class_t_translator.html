Detailed Description

A utility class that facilitates translation. A Translator is used to transform a string that contains markup and translates it for consumption via a display method. Multiple display targets are supported via an OutputPolicy. FOutputToGfxPolicy is a policy that simply converts from UE3's markup to Scaleform GFx markup. One could also implement a CanvasPolicy; such a policy would render the output to an FCanvas.
