◆ GMouseLockClientRect

RECT GMouseLockClientRect = { 0,0,0,0 }

This is client RECT for the window we most recently locked the mouse to (in screen coordinates). Note that it may not exactly match the rect returned by GetClipCursor, since that rect will be constrained to the desktop area, while our client rect will not.

The mouse lock rect is a shared system resource, so we should not try to unlock a mouse if some other window wants it locked.
