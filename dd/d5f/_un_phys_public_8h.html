)

Take an FKAggregateGeom and cook the convex hull data into a FKCachedConvexData for a particular scale. This is a slow process, so is best done off-line if possible, and the results stored.

Parameters

ConvexElemsThe input aggregate geometry. Each convex element will be cooked.
Scale3DThe 3D scale that the geometry should be cooked at.
