#define STATICMESH_VERSION   18

Static mesh build version, forces any versions older than this to be rebuilt on load.
Warning: Incrementing this effectively requires a content resave! Do not use this version for native serialization backwards compatibility, instead use VER_LATEST_ENGINE/VER_LATEST_ENGINE_LICENSEE.
