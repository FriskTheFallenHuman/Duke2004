Detailed Description

Octree containing NavigationPoint/ReachSpec bounding boxes for quickly determining things like whether a given location is on the path network

We deliberatly don't serialize its object references as the octree won't be the only reference and the AddReferencedObject interface doesn't support NULLing out references anyways.
