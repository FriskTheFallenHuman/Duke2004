Detailed Description

FTerrainBrowserBlock

+--------------—+----------------------------—+

  • [V] [L] [W] [S] | [X] [TN0] [TN1] Name [I] | +--------------—+----------------------------—+

V = View icon L = Lock icon W = Wireframe overlay icon S = Solid overlay icon X = Open/Close icon TN0 = Thumbnail0 - Material snapshot or identifying icon TN1 = Thumbnail1 - Displacement map (if present) I = Information icon
