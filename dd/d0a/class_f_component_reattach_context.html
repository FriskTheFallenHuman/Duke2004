Detailed Description

Detaches a component for the lifetime of this class.

Typically used by constructing the class on the stack: { FComponentReattachContext ReattachContext(this); // The component is removed from the scene here as ReattachContext is constructed. ... } // The component is returned to the scene here as ReattachContext is destructed.
