◆ JumpTargets

INT FNestInfo::JumpTargets[FIXUP_MAX]

Actual bytecode locations for the corresponding jump type (i.e. FIXUP_*) These values are used by PopNest to replace previously emitted placeholder locations in cases where the a jump location was required, but wasn't yet known. The values are indexes into the node's Script array.
