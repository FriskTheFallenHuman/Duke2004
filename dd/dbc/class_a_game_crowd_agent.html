◆ ShouldPerformCrowdSimulation()

UBOOL AGameCrowdAgent::ShouldPerformCrowdSimulation ( FLOAT  DeltaTime)
virtual

This will allow subclasses to implement specialized behavior for whether or not to actually simulate. Example: You have hundreds of crowd agents and not all can be seen. So doing a distance check before simulating them could save CPU. (distance check in stead of LastRender as you might want them moving before the viewer sees them).

This will allow subclasses to implement specialized behavior for whether or not to actually simulate. Base behavior is to reduce update frequency based on how long since an agent has been visible.
