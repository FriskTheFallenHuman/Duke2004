Detailed Description

template<UINT NumInlineElements, typename SecondaryAllocator = FDefaultAllocator>
class TInlineAllocator< NumInlineElements, SecondaryAllocator >

The inline allocation policy allocates up to a specified number of bytes in the same allocation as the container. Any allocation needed beyond that causes all data to be moved into an indirect allocation. It always uses DEFAULT_ALIGNMENT.
