◆ StereoizedDrawNullTarget()

FSurfaceRHIRef StereoizedDrawNullTarget ( const FSurfaceRHIRef &  altBuffer)

Returns a NULL target if we're not running in stereo. If running in stereo return the specified altBuffer. altBuffer may not be the correct size, and so could trigger warnings or errors in the d3d runtime. However, by providing a buffer that will be unused, we're cueiing the stereo driver that this call should be stereoized.
