Detailed Description

template<UBOOL bEnableDepthWrite = TRUE, ECompareFunction DepthTest = CF_LessEqual>
class TStaticDepthState< bEnableDepthWrite, DepthTest >

A static RHI depth state resource. TStaticDepthStateRHI<...>::GetStaticState() will return a FDepthStateRHIRef to a depth state with the desired settings. Should only be used from the rendering thread.
