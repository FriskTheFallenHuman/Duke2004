◆ ClickedPropertyItem()

UBOOL WxPropertyControl::ClickedPropertyItem ( wxMouseEvent &  Event)
protectedvirtual

Called when an property window item receives a left-mouse-button press which wasn't handled by the input proxy. Typical response is to gain focus and (if the property window item is expandable) to toggle expansion state.

Parameters
Eventthe mouse click input that generated the event
Returns
TRUE if this property window item should gain focus as a result of this mouse input event.

Reimplemented in WxCustomPropertyItem_LightmassMaterialParameter, WxCustomPropertyItem_MIC_Parameter, WxCustomPropertyItem_MaterialInstanceTimeVaryingParameter, WxItemPropertyControl, and WxCustomPropertyItem_ConditionalItem.
