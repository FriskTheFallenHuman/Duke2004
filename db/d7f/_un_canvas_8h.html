Render string using both a font and a material. The material should have a font exposed as a parameter so that the correct font page can be set based on the character being drawn.

Parameters
Canvas- valid canvas for rendering tiles
StartX- starting X screen position
StartY- starting Y screen position
XScale- scale of text rendering in X direction
YScale- scale of text rendering in Y direction
Text- string of text to be rendered
Font- font containing texture pages of character glyphs
MatInst- material with a font parameter
FontParam- name of the font parameter in the material
Returns
total size in pixels of text drawn
