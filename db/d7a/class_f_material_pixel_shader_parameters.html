Set the material shader parameters which depend on the mesh element being rendered.

Parameters
PixelShader- The pixel shader to set the parameters for.
View- The view that is being rendered.
Mesh- The mesh that is being rendered.
bBackFace- True if the backfaces of a two-sided material are being rendered.

Set local transforms for rendering a material with a single mesh

Parameters
MaterialRenderContext- material specific info for setting the shader
LocalToWorld- l2w for rendering a single mesh
