Detailed Description

Property windows which are meant to be stand-alone windows must sit inside of a WxPropertyWindowFrame. Floating property windows can contain an optional toolbar, and can be locked to paticular properties. WxPropertyWindowFrame also manages whether or not a property window can be closed.
