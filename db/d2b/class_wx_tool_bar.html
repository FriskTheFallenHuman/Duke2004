Detailed Description

This class inherits from wxToolBar in order to provide tooltip functionality beyond what is provided in the wxWidgets implementation. Tooltips for tools on wxToolBars display properly as long as the window with the toolbar has focus, but will not display if the window does not, contrary to how the rest of the tooltips provided by wxWidgets function. This is due to wxWidgets handling the tooltips of toolbar tools in a different manner that appears to be buggy. In lieu of modifying the wxWidgets source and causing future version troubles, WxToolBar provides a low-impact solution to the problem until such time wxWidgets corrects the behavior by responding to the event generated when the mouse enters/exits a tool on the toolbar and setting the toolbar's tooltip itself accordingly (because tooltips set via SetToolTip work properly, even without focus). Additionally, the class disables the normal tooltip method for tools by forcibly passing in wxTB_NO_TOOLTIPS to the wxToolBar constructor, such that the two systems do not conflict.
