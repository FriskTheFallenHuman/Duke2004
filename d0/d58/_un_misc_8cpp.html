When a generated .ini is written out it will store the timestamps of the files it was generated from. This way whenever the Default__.inis are modified the Generated .ini will view itself as outdated and regenerate it self.

Outdatedness also can be affected by commandline params which allow one to delete all .ini, have automated build system etc.
