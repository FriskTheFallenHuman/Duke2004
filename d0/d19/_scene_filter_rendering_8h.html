Uses a Gaussian blur to filter a subregion of the filter buffer.

Parameters
View- The view that is being used in rendering
ViewSizeX- The view width in pixels to scale the radius.
SizeX- The width of the subregion to blur.
SizeY- The height of the subregion to blur.
KernelRadius- The radius of the Gaussian kernel.
KernelScale- Extra scale on the filter kernel (does not influence the filter weights)
FilterColorIndex- Index of the render target to apply the Gaussian blur to
SampleMaskMin- Minimum values used to clamp the filter samples. For no clamping use FVector2D(-1.f, -1.f)
SampleMaskMax- Maximum values used to clamp the filter samples. For no clamping use FVector2D(2.f, 2.f)
