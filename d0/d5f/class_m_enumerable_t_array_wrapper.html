Detailed Description

template<class ElementWrapperType, class ElementType>
class MEnumerableTArrayWrapper< ElementWrapperType, ElementType >

MEnumerableTArrayWrapper - wrapper for Unreal TArray to provide an IEnumerable interface so the data it can be bound to a WPF ItemsSource. @templateparam ElementWrapperType - managed class taking a reference to the TArray element end exposing the variables as properties @templateparam ElementType - native struct stored in the TArray.
