◆ appShowSplash()

void appShowSplash ( const TCHAR *  InSplashName)

Displays a splash window with the specified image. The function does not use wxWidgets. The splash screen variables are stored in static global variables (SplashScreen*). If the image file doesn't exist, the function does nothing.

Parameters
InSplashNameName of splash screen file (and relative path if needed)
