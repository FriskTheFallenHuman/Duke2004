Detailed Description

WxDockableWindowMenuHandler

Implements support for managing a 'Window' menu in a menu bar that updates automatically in response to dockable windows being opened, closed, docked and floated. In general, this serves as a tool for the user to unhide floating windows that have been hidden (by pressing the close button.)
