◆ ALLOW_LIGHTMAP_SAMPLE_DEBUGGING

#define ALLOW_LIGHTMAP_SAMPLE_DEBUGGING   0

Set to 1 to allow selecting lightmap texels by holding down T and left clicking in the editor, And having debug information about that texel tracked during subsequent lighting rebuilds. Be sure to set the define with the same name in Lightmass!
