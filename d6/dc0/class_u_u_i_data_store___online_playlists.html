Callback for retrieving a textual representation of natively serialized properties. Child classes should implement this method if they wish to have natively serialized property values included in things like diffcommandlet output.

Parameters
out_PropertyValuesreceives the property names and values which should be reported for this object. The map's key should be the name of the property and the map's value should be the textual representation of the property's value. The property value should be formatted the same way that UProperty::ExportText formats property values (i.e. for arrays, wrap in quotes and use a comma as the delimiter between elements, etc.)
ExportFlagsbitmask of EPropertyPortFlags used for modifying the format of the property values
Returns
return TRUE if property values were added to the map.
