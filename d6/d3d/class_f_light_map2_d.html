static

Allocates texture space for the light-map and stores the light-map's raw data for deferred encoding. If the light-map has no lights in it, it will return NULL. RawData and SourceQuantizedData will be deleted by this function.

Parameters
LightMapOuter- The package to create the light-map and textures in.
RawData- The raw light-map data to fill the texture with.
SourceQuantizedData- If the data is already quantized, the values will be in here, and not in RawData.
Material- The material which the light-map will be rendered with. Used as a hint to pack light-maps for the same material in the same texture. Can be NULL.
Bounds- The bounds of the primitive the light-map will be rendered on. Used as a hint to pack light-maps on nearby primitives in the same texture.
InPaddingType- the method for padding the lightmap.
LightmapFlags- flags that determine how the lightmap is stored (e.g. streamed or not)
