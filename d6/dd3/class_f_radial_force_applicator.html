Called to compute a number of forces resulting from a force field.

Parameters
PositionsArray of input positions (in world space)
PositionStrideNumber of bytes between consecutive position vectors
VelocitiesArray of inpute velocities
VelocityStrideNumber of bytes between consecutive velocity vectors
OutForceOutput array of force vectors, computed from position/velocity, forces are added to the existing value
OutForceStrideNumber of bytes between consectutiive output force vectors
CountNumber of forces to compute
Scalefactor to apply to positions before use(ie positions may not be in unreal units)
PositionBoundingBoxBounding box for the positions passed to the call
Returns
