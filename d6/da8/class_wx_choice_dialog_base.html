Detailed Description

Base class for choice-based dialogs. Should not be used on its own, but instead inherited from to provide choice-based dialogs with potentially dramatically different main panels (therefore has pure virtual destructor to prevent instantiation). Inheriting classes should implement their primary "window" as a new wxPanel and then send it to WxChoiceDialogBase::ConstructWidgets in order to properly build the dialog. ConstructWidgets will place the panel above the appropriate buttons with a sizer. See WxChoiceDialog as an example.
