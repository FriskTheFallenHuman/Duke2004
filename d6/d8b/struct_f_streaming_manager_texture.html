◆ GetWantedMipsForOrphanedTexture()

INT FStreamingManagerTexture::GetWantedMipsForOrphanedTexture ( FStreamingTexture &  StreamingTexture,
FLOAT &  Distance 
)
protected

Fallback handler to catch textures that have been orphaned recently. This handler prevents massive spike in texture memory usage. Orphaned textures were previously streamed based on distance but those instance locations have been removed - for instance because a ULevel is being unloaded. These textures are still active and in memory until they are garbage collected, so we must ensure that they do not start using the LastRenderTime fallback handler and suddenly stream in all their mips - just to be destroyed shortly after by a garbage collect.
