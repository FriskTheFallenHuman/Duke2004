◆ PatchVertexStreamOffsetsToBeUnique()

static void PatchVertexStreamOffsetsToBeUnique ( FVertexDeclarationElementList &  Elements)
static

Patches the declaration so vertex stream offsets are unique. This is required for e.g. GeForce FX cards, which don't support redundant offsets in the declaration. We're unable to make that many vertex elements point to the same offset so the function moves redundant declarations to higher offsets, pointing to garbage data.
