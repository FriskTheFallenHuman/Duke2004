IsOneWayEdge

  • indicates whether or not this edge is a non-shared edge, or an edge which could be added to one adjacent poly and not the other (e.g. special edges that only work in one direction should return TRUE) NOTE: By default edges that are one way are ignored during obstacle mesh creation
