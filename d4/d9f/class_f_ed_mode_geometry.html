◆ GeomObjects

TArray<FGeomObject*> FEdModeGeometry::GeomObjects
protected

Custom data compiled when this mode is entered, based on currently selected brushes. This data is what is drawn and what the LD interacts with while in this mode. Changes done here are reflected back to the real data in the level at specific times.
