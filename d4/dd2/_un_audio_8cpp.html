◆ FixListenerScopeRadius()

static void FixListenerScopeRadius ( AAmbientSoundSpline *  AmbientSoundSpline)
static

In all AAmbientSoundSpline (and inherited classes) actors MaxRadius (in Attenuation group) must be smaller than ListenerScopeRadius. Otherwise the ambient sound will occur rapidly with relatively loud volume. A user should be responsible for the proper setup, but since it may be hard to track all used sound nodes, it is done automatically.
