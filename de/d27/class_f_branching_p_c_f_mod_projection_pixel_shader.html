Detailed Description

template<class BranchingPCFPolicy>
class FBranchingPCFModProjectionPixelShader< BranchingPCFPolicy >

A pixel shader for projecting an object's shadow buffer onto the scene. Attenuates shadow based on distance and modulates its color. For use with modulated shadows.
