Draws the primitive's dynamic decal elements. This is called from the rendering thread for each frame of each view. The dynamic elements will only be rendered if GetViewRelevance declares dynamic relevance. Called in the rendering thread.

Parameters
PDIThe interface which receives the primitive elements.
ViewThe view which is being rendered.
InDepthPriorityGroupThe DPG which is being rendered.
bDynamicLightingPassTRUE if drawing dynamic lights, FALSE if drawing static lights.
bDrawOpaqueDecalsTRUE if we want to draw opaque decals
bDrawTransparentDecalsTRUE if we want to draw transparent decals
bTranslucentReceiverPassTRUE during the decal pass for translucent receivers, FALSE for opaque receivers.
