TID_Visibility_Off 

This flag indicates that the current terrain 'quad' is not visible...

TID_OrientationFlip 

This flag indicates that the 'quad' should have it's triangles 'flipped'

TID_Unreachable 

This flag indicates the patch is unreachable.
Used to tag areas the player(s) should not be able to reach but are sitll visible (tops of mountains, etc.).

TID_Locked 

This flag indicates the patch is locked. The vertices can no longer be edited.
