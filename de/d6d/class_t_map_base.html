Detailed Description

template<typename KeyType, typename ValueType, UBOOL bInAllowDuplicateKeys, typename SetAllocator = FDefaultSetAllocator>
class TMapBase< KeyType, ValueType, bInAllowDuplicateKeys, SetAllocator >

The base class of maps from keys to values. Implemented using a TSet of key-value pairs with a custom KeyFuncs, with the same O(1) addition, removal, and finding.
