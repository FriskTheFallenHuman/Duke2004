Accumulates another transform with this one, with a blending weight

Let SourceAtom = Atom * BlendWeight Rotation is accumulated multiplicatively (Rotation = SourceAtom.Rotation * Rotation). Translation is accumulated additively (Translation += SourceAtom.Translation) Scale is accumulated multiplicatively (Scale *= SourceAtom.Scale)

Note: Rotation will not be normalized! Will have to be done manually.
