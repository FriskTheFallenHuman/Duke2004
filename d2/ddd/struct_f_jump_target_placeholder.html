Detailed Description

A code jump-target address placeholder. Whenever a code jump is emitted where the destination of the jump has yet to be determined, a placeholder is inserted into the bytecode (e.g. the location to jump to if an 'if' statement evaluates to false.

FJumpTargetPlaceholder is used to track the location of the placeholder so that it can be fixed up later.
