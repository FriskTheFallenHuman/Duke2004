UBOOL FNameEntry::HasAnyFlags ( EObjectFlags  FlagsToCheck) const

Used to safely check whether any of the passed in flags are set. This is required as EObjectFlags currently is a 64 bit data type and UBOOL is a 32 bit data type so simply using GetFlags() & RF_MyFlagBiggerThanMaxInt won't work correctly when assigned directly to an UBOOL.
