◆ PACKAGE_INDEX

#define PACKAGE_INDEX   INT

Index into a ULnker's ImportMap or ExportMap. Values greater than zero indicate that this is an index into the ExportMap. The actual array index will be (PACKAGE_INDEX - 1).

Values less than zero indicate that this is an index into the ImportMap. The actual array index will be (-PACKAGE_INDEX - 1)
