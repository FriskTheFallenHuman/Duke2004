virtual void IConsoleManager::UnregisterConsoleVariable ( IConsoleVariable *  CVar)
pure virtual

Deactivated the use of the console variable but not actually removes it as some pointers might still reference it. If the variable is registered again with the same type this object is reactivated. This is good for DLL unloading.

Parameters
CVarcan be 0 then the function does nothing
