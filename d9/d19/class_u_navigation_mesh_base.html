Detailed Description

UNavigationMeshBase - the primary container of the mesh itself.
Meshes are used for multiple purposes (obstaclemesh/navigationmesh/etc.) Meshes contain a pool of verts, a pool of polys, and a pool of edges that represent walkable connections between polys
