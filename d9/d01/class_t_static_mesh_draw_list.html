Detailed Description

template<typename DrawingPolicyType>
class TStaticMeshDrawList< DrawingPolicyType >

A set of static meshes, each associated with a mesh drawing policy of a particular type.

Parameters
DrawingPolicyType- The drawing policy type used to draw mesh in this draw list.
