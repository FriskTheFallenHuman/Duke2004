◆ GetMaxIntegral()

virtual FLOAT FFogVolumeDensitySceneInfo::GetMaxIntegral ( ) const
pure virtual

Returns an estimate of the maximum integral value that will be calculated by this density function. This will be used to normalize the integral on platforms that it has to be stored in fixed point. Too large of estimates will increase aliasing, too small will cause the fog volume to be clamped.

Implemented in FFogVolumeConstantDensitySceneInfo, FFogVolumeConstantHeightDensitySceneInfo, FFogVolumeLinearHalfspaceDensitySceneInfo, FFogVolumeSphericalDensitySceneInfo, and FFogVolumeConeDensitySceneInfo.

UBOOL FFogVolumeDensitySceneInfo::bAffectsTranslucency
