◆ DelayedCrossLevelFixupMap

TMultiMap<FGuid, FDelayedCrossLevelRef> FCrossLevelReferenceManager::DelayedCrossLevelFixupMap

Map of Guids used by cross level references to locations in other objects that attempted to point to this object, but the object wasn't already loaded. This will allow properties to be automatically fixed up to point to this object after the level is loaded.

List of objects that are being pointed to by external objects. This is used so when an object is unstreamed, it NULLs out pointers pointing to it, as well as readies them to be streamed in again using the above DelayedCrossLevelFixup map
