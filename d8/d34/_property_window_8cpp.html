static FObjectPropertyNode * NotifyFindObjectItemParent ( WxPropertyControl *  InNode)
static

Searches for nearest WxObjectsPropertyControl ancestor, returning NULL if none exists or InNode is NULL. Different from the WxPropertyControl version in that an input of NULL is acceptable, and that calling with an InNode of type WxObjectsPropertyControl returns the enclosing WxObjectsPropertyControl, not the input itself.
