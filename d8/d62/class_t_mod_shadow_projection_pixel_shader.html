Detailed Description

template<class LightTypePolicy, class UniformPCFPolicy>
class TModShadowProjectionPixelShader< LightTypePolicy, UniformPCFPolicy >

TModShadowProjectionPixelShader - pixel shader used with lights casting modulative shadows Attenuation is based on light type so the modulated shadow projection is coupled with a LightTypePolicy type
