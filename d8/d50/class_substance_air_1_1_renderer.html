◆ push() [1/2]

void SubstanceAir::Renderer::push ( SubstanceAir::FGraphInstance *  graph)

Push graph instance current changes to render The current GraphInstance state is used to enqueue a render job: input values and flags, output flags.

Only the outputs of the graph flagged as dirty (outdated) are rendered (use OutputInstance::flagAsDirty() to force rendering).
When a GraphInstance is pushed for the first time, all outputs are considered dirty (and therefore are computed later).

Note
Note that the rendering process is not started by this command but with the run() method.

◆ setCallbacks()

void SubstanceAir::Renderer::setCallbacks ( Callbacks *  callbacks)

Set user callbacks.

Parameters
callbacksPointer on the user callbacks concrete structure instance that will be used for the renderer callbacks call. Can be NULL pointer: callbacks no more used.
Warning
A set callback instance is used by all computation created by run() calls, even if another callback instance is set just after. Then take care of Callbacks instance lifetime (to avoid issues, should be deleted AFTER Renderer instance).
