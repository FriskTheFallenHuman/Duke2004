Detailed Description

AnimNodeSlot : AnimNodeSequence pool

If you use this system, it will remove current AnimNodeSequence of (child 1-(N-1) of N), and use one on demand from pool That way, we don't have to keep many AnimNodeSequence in memory but use it when it only needed This pool is shared between any pawn
