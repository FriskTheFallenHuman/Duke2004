Detailed Description

This class processes a buffer of text, expanding macro expressions. Constructed with the name of the package; this gives an initial search path for included files.

FString FMacroProcessingFilter::ParseNextParam ( )
private

Retrieves the next parameter from a macro expansion. It is assumed that we are just past the opening left parenthesis or comma. This function will skip leading whitespace and assemble the parameter until it sees a top-level (to it) comma or closing parenthesis. It will expand macros as they are encountered.

Returns
the next parameter for the current macro
