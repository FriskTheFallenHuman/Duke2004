Detailed Description

Implementation of realtime garbage collector.

The approach is to create an array of DWORD tokens for each class that describe object references. This is done for script exposed classes by traversing the properties and additionally via manual function calls to emit tokens for native only classes in the StaticConstructor. A third alternative is a AddReferencedObjects callback per object which is used to deal with object references from types that aren't supported by UnrealScript's type system. interface doesn't make sense to implement for.
